# Gmail inbox `gmail`

Two-minute rule: reply, archive, or turn it into a task — then it leaves the inbox.
Anything bigger becomes a Todoist task; don't let it live in here.

# Todoist inbox `todoist`

Every loose item gets a project and a date, or it gets deleted.
No-action keepers go to Someday, not back on the list.

# Clean Downloads folder `downloads`

File the keepers, bin the installers and duplicates.
When in doubt you can download it again — let it go.

Generate fresh, store in the manager, never reuse.

# Descale the kettle `descale-kettle`

Fill to max with half water, half white vinegar.
Boil, then leave to sit for an hour.
Tip out, rinse twice, boil once with clean water before use.

# Clean the oven `clean-oven`

Take the racks out and soak them in the bath.
Spray the cavity, leave it 20 minutes, then wipe down.

# Test the smoke alarms `test-smoke-alarms`

Hold the test button until it sounds. Swap any battery that hesitates.

# Wash the windows `wash-windows`

# Service the bicycle `service-bicycle`

Tyre pressure, brake pads, chain oiled, gears shifting clean.
Spin each wheel and listen — anything ticking gets looked at.

# Rotate the mattress `rotate-mattress`

Head-to-foot this time; flip top-to-bottom next time. Alternate each round.
Hoover the surface while it's exposed.

# Flush the water heater `flush-water-heater`

Kill the power first, then connect a hose to the drain valve.
Run it until the water comes out clear, not cloudy, before refilling.
